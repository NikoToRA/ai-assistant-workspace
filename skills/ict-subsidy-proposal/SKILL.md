# ICT補助金 提案書スキル

病院・医療機関向けに厚生労働省 業務効率化ICT補助金を活用したコエレク導入提案書を作成する。

**デザイン基盤:** `skills/wonder-drill-slides/` を参照（theme.css・template.md）

---

## 補助金の基礎知識

### 制度概要（令和8年度）

- **正式名称:** 医療機関等における業務効率化・生産性向上支援事業
- **根拠:** 医政発0213第22号（令和8年2月13日）
- **補助率:** 5分の4（80%）
- **補助上限:** 8,000万円/施設
- **対象:** 病院（ベースアップ評価料の届出済みが要件）

### 補助金の要件

| 要件 | 内容 |
|------|------|
| 対象機関 | ベースアップ評価料の届出済み病院 |
| ICT機器の導入 | 生成AIを活用した文書自動作成支援等 |
| 附随する研修 | ICT機器導入に附随する訓練（セット必須） |
| 業務効率化計画 | 目標・計画書の策定・提出（5〜6月） |
| 成果の報告 | 3年後の評価・報告書提出 |

**重要:** 研修単体・ICT機器のみでは申請不可。セットで申請する必要がある。

### 申請スケジュール

- **3〜4月:** 都道府県が所要見込額を決定
- **5〜6月:** 申請書・業務効率化計画 提出 ← ここが締め切り
- **7月〜:** 厚生労働省が選定・採択決定
- **採択後:** ICT機器導入・研修実施
- **3年後:** 成果評価・報告書提出

### 返還リスク

厚生労働省 実施要綱より：「評価において成果が認められなかった場合には補助金の返還を求める場合がある」

返還リスクの主な原因:
- 初年度に目標値・評価指標を適切に設定しなかった
- 年次の進捗記録・エビデンスが残っていない
- 3年後の評価表・報告書を準備できなかった

## 提案書の標準構成（7枚スライド）

| # | スライドタイトル | 内容 |
|---|----------------|------|
| 1 | 表紙 | 病院名・補助金名・提案タイトル・日付 |
| 2 | 補助金 × コエレク | 申請構造・コエレク概要 |
| 3 | 料金表 | アカウント費・研修メニュー |
| 4 | 3ヵ年サポートプラン | 3年コスト全体像・補助後実質負担 |
| 5 | 評価サポートが必要な理由 | 返還リスク・評価サポート内容 |
| 6 | 申請要件と流れ | 要件チェックリスト・スケジュール |
| 7 | 次のステップ | Wonder Drillが提供できること・確認事項 |

状況に応じてスライドを追加・省略してよい。

## 手順

### Step 1: ヒアリング

病院側から以下を確認する:
- 病院名・担当者名
- ベースアップ評価料の届出状況
- 導入希望アカウント数（対象スタッフ数）
- カルテ記載形式（SOAP等）・院内ルール
- 補助金申請の意向（積極的 / 検討中 / 不明）
- 特に強調したい訴求ポイント

### Step 2: 提案書ファイルを作成

1. `skills/ict-subsidy-proposal/template.md` を `notes/<病院名>-proposal.md` にコピー
2. 以下のプレースホルダーを実際の値に置換する:

| プレースホルダー | 置換内容 | 例 |
|----------------|---------|-----|
| `{{病院名}}` | 病院名（「さま」なし） | `守山病院` |
| `{{年月}}` | 提案日の年月 | `2026年3月` |

3. アカウント数が複数の場合は料金表・3ヵ年表の数値を計算し直す（計算式は下記）
4. 必要に応じてCTAスライドの文言をカスタマイズ

### Step 3: PDF生成

```bash
npx @marp-team/marp-cli notes/<病院名>-proposal.md \
  --theme skills/wonder-drill-slides/theme.css \
  --pdf \
  --allow-local-files
```

### Step 4: Discordに送信

```
MEDIA:notes/<病院名>-proposal.pdf
```

## 多アカウントの見積り計算

アカウント数 N の場合（初年度・税別）:

- コエレク: 299,800 × N 円（おまとめBoxは1台/人）
- 研修: 150,000 × 3本 = 450,000円（人数制限なし・定額）
- 評価サポート: 800,000円（任意）

**初年度合計:** (299,800 × N) + 450,000 + 800,000 円
**補助後実質:** 上記 × 0.2
